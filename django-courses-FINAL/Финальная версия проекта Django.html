К этой лекции присоединён архив с финальной версией проекта Django.

Но я крайне рекомендую вам создавать этот проект самостоятельно от начала до конца


Предварительные требования:

  • Python 3.11

  • pipenv (install using pip install pipenv)


  1. Создать папку .venv в папке проекта

    mkdir .venv


  2. Установить зависимости

    pipenv install


  3. Активировать виртуальную среду

    pipenv shell


  4. Запустить сервер Django

    python manage.py runserver


  5. Открыть веб проект Django в веб браузере

    http://127.0.0.1:8000


  6. Логин и пароль администратора для административной панели http://127.0.0.1:8000/admin
