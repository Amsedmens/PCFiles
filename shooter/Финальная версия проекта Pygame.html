К этой лекции присоединён архив с финальной версией проекта Pygame.

Но я крайне рекомендую вам создавать этот проект самостоятельно от начала до конца


Предварительные требования:

  • Python 3.11

  • pipenv (install using pip install pipenv)


  1. Создать папку .venv в папке проекта

    mkdir .venv


  2. Установить зависимости

    pipenv install


  3. Активировать виртуальную среду

    pipenv shell


  4. Запустить игру shooter

    python main.py


  5. Запустить игру rectangle

    python rectangle.py
